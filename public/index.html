Trapezoidal Rule

V = (h/2) × [A₀ + 2A₁ + 2A₂ + ... + Aₙ]

✓ Advantages: Simple computation, works for any number of sections, good for linear changes

⚠ Considerations: Less accurate for curved surfaces

Pyramid Rule (Simpson's 1/3)

V = (h/3) × [A₀ + 4A₁ + 2A₂ + 4A₃ + ... + Aₙ]

✓ Advantages: Parabolic approximation, better accuracy

⚠ Requirements: Odd number of sections • Condition: An/An-1 ≤ 0.5

Simpson's 3/8 Rule

V = (3h/8) × [A₀ + 3A₁ + 3A₂ + 2A₃ + ...]

✓ Advantages: Cubic polynomial, most accurate method, best for smooth curves

✓ Best For: Highest precision results and complex geometries
